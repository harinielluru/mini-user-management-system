# Mini User Management System

A full-stack web application that provides secure user authentication, role-based access control, and admin user management. This project demonstrates backend and frontend integration using modern web technologies.

The Mini User Management System allows users to register, log in securely, and view their own profiles. Administrators have additional privileges to view all users and manage their account status. The system uses JWT-based authentication and role-based authorization to ensure security.
